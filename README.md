# CulturaBuilder Website

Site oficial do CulturaBuilder MCP - Framework que estende Claude Code com comandos especializados e smart personas.

## 🌐 Site Online

**URL:** https://prldlkcd.manus.space

## 🚀 Sobre o CulturaBuilder

O CulturaBuilder é um framework Python que estende Claude Code com:

- **16 comandos especializados** (/cb:implement, /cb:build, /cb:analyze, etc.)
- **6+ smart personas** (architect, frontend, backend, analyzer, security, scribe)
- **Integração MCP** (Context7, Sequential, Magic, Playwright)
- **Token optimization** para conversas mais longas
- **Task management** com smart routing

## 📦 Instalação do CulturaBuilder

```bash
# Instalar via PyPI
pip install CulturaBuilder

# Executar configurador
python3 -m CulturaBuilder install

# Testar no Claude Code
/cb:explain
```

## 🛠️ Desenvolvimento do Site

Este site foi desenvolvido com:

- **React** + **Vite**
- **Tailwind CSS** + **shadcn/ui**
- **Framer Motion** para animações
- **Lucide Icons**

### Executar Localmente

```bash
# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev

# Build para produção
npm run build
```
